## 文档书写要求

1. 生成文档

`php art doc app/user/登录接口`

需要事先在`doc`目录创建好文档目录，同时，需要在`config.json`中按要求配置

2. 语法

请严格参照 markdown 语法来书写

3. 条目

- 接口名称（必须）
- 接口地址（必须）
- 请求方式（必须）GET/POST/DELETE/PUT/OPTIONS
- 请求参数（必须）必填参数在前，非必填灾后
- 响应结果（必须）json格式
- 错误示例（非必须）
